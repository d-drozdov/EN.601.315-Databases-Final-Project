## Local Development
### Requirements:
- Python 3.10
- PIP
- Docker

### Setup
1. Clone the repo
2. Run `pip install -r requirements.txt`
3. Run `docker compose up -d`
4. To connect to this db use the following connection string: `postgresql://postgres:postgres@localhost:5432/postgres`
